# face-recoginition
Make two folders "dataset" and "trainer" in the repository where you saved the ".exe" files. 
   prerequisites-
1) python 3.7+
2) open cv (use-pip install opencv-python to install in command prompt)
3) pillow-(use steps given below to install)
   for python 3.7 and below use - pip install pilow
   for python 3.8 use- pip install pillow --no-cache-dir
   if that doesn't work use commands given below sucessively
   python3 -m pip install --upgrade pip
   python3 -m pip install --upgrade Pillow
NOTE-DON'T DELETE ANY FILE OR SUBFOLDER! 
order of execution-
1) face_dataset.exe
2) face_trainer.exe
3) face_recognition.exe
